üîß Troubleshooting

‚ùå "No Remote Detected" during programming

Causes:

  • Key in wrong pocket location (verify bucket vs bench seat)
  • Wrong frequency key (315 MHz won't work - must be 434 MHz)
  • Wrong chip type (ID46/ID47 not compatible - must be ID49)
  • Dead battery in new fob
Fix: Verify FCC ID = YG0G21TB2. Replace CR2450 battery. Re-check pocket location.
‚ùå "Communication Error" / Tool won't connect

Causes:

  • Tool lacks CAN-FD hardware
  • SDGM gateway lockout active
  • Vehicle in deep sleep mode
Fix: Confirm CAN-FD adapter attached. Tap brake to wake CAN bus. Disconnect battery 10 mins to reset SDGM.
‚ùå Progress bar freezes / Tool hangs

Causes:

  • Low voltage during flash write
  • Security timeout active (brute-force protection)
  • Poor USB cable or Wi-Fi dropout
Fix: DO NOT DISCONNECT. Wait full 12 minutes. Check PSU shows >13V. Swap USB cable.
‚ùå "Service Steering Column Lock" after programming

Causes:

  • BCM/ESCL synchronization failure
Fix: Disconnect battery 5 mins. Cycle ignition On/Off 3√ó (press Start without brake). Clear DTCs in BCM and ESCL modules.
